***FUNCIONALIDADES***

sing up = o usuário pode fazer o seu cadastro com username e 
senha, após isso ele já é redirecionado para o profile/create.
Se um usuário já cadastrado tentar cadastrar seu username novamente
o programa retorna uma mensagem informando que o usuário já existe.

login = usuário que já tem profile faz login e é redirecionado
para sua dashboard onde pode fazer logout, ver detalhes de seu 
perfil ou ver sua lista de macros planner.

logout = ao fazer logout o usuário é redirecionado a página
de login e singup, homepage.

profile/details = o usuário visualiza os detalhes do profile do 
usuário logado e da opção de logout, profile details, lista de 
macros planner, e as opções de editar perfil ou deleta-lo.

profile/update = ao clicar em update o usuário pode alterar todas
suas informações ***BUG*** exceto editar seu genêro, após isso ele
é redirecionado ao profile/details.

profile/delete = ao clicar no botão delete o sistema retorna um alert
perguntando se o usuário quer realmente deletar seu perfil ***BUG***
o perfil é deletado clicando em ok ou em cancelar (possivel solução
dar um commit=False ao clicar no btn deletar e só confirmar a partir da
resposta do alert). Após o usuário deletar seu profile ele é redirecionado
ao profile create.

profile/create = ao criar o profile onde o usuário pode adicionar todos os 
seus dados ***BUG*** exceto o gender. depois disso ele é redirecionado para
o profile/details onde ele pode fazer logout, profile details, visualizar
macros planner list, editar ou deletar seu profile.


macros_planner/list = caso o usuário tenha algum macros planner, eles são listados
e o usuário tem a opção de ver os seu detalhes ou exclui-los, ele também pode criar
um novo macros planner.

macros_planner/create = ao clicar em criar um macros planner, o usuário é redirecionado
para o GeneralInfo, onde ele pode ou não informar um objetivo, deve informar uma data 
de inicio e de fim e um número de semanas, que aceita como valor mínimo 1. Após criar o
macros planner o usuário é redirecionado ao macros_planner/details onde ele visualiza o 
GeneralInfo e tem opções de editar este objeto, também é criado automáticamente uma macros
sheet. O usuário tem opção de editar ou deletar essa macros sheet, ou criar novas macros sheet.

general_info/update = o usuário pode atualizar todas as informações gerais, e é 
redirecionado para general_info/details onde ele tem as opções de editar o GeneralInfo, 
ou de ver os detalhes do macros planner, e também fazer logout.

macros_sheet/delete = ao clicar nessa opção o usuário deleta a macros sheet e é redirecionado 
aos detalhes do macros planner.


macros_sheet/create = o usuário pode informar os valores para cada macro nutriente, o programa
aceita como valor mínimo 1. após clicar no botão de create, o usuário é redirecionado para os
detalhes da macros sheet, onde ele pode edita-la deleta-la ou visualizar os detalhes do 
macros planner.

macros_sheet/update = após atualizar o usuário é redirecionado ao macros_sheet/details.

macros_planner/delete = ao clicar em excluir macros planner o programa NÃO informar nenhum alert
pedindo confirmação, apenas exclui o macros planner e redireciona o usuário para o macros planner
list.
