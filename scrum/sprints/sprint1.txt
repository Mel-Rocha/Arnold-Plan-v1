Sprint 1: Módulo de Autenticação

Cadastro por e-mail:

Implementar a funcionalidade de cadastro de usuários usando o e-mail.
Validar os campos do formulário de cadastro.
Armazenar as informações do usuário no banco de dados.
Cadastro social (Google):

Permitir que os usuários se cadastrem usando suas contas do Google.
Integrar a API do Google para autenticação e obtenção dos dados do usuário.
Login por e-mail:

Implementar a funcionalidade de login usando o e-mail e senha.
Verificar a autenticidade das credenciais fornecidas.
Gerenciar a sessão do usuário após o login.
Login social (Google):

Permitir que os usuários façam login usando suas contas do Google.
Integrar a API do Google para autenticação.
Gerenciamento de sessão (Lembre-me):

Implementar a opção "Lembre-me" para manter a sessão do usuário ativa por um período prolongado.
Armazenar as informações de login em um cookie ou token seguro.
Automação de e-mails:

Configurar e implementar o envio automático de e-mails para os usuários, como confirmação de cadastro e redefinição de senha.
Recuperar senha:

Implementar a funcionalidade de recuperação de senha para os usuários.
Enviar um e-mail com um link seguro para redefinir a senha.
Personalizar perfil:

Permitir que os usuários editem informações do perfil, como foto, nome e senha.
Implementar as alterações no banco de dados.
